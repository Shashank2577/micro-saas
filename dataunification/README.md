# DataUnification

A data unification and integration platform that consolidates data from multiple disparate sources. Enable teams to create unified customer views, reconcile data across systems, and sync changes in real-time.

## Features
- Multi-source data connectors (databases, APIs, files, cloud storage)
- Visual schema mapping and transformation (source field -> target field)
- Data reconciliation engine
- Identity resolution
- Real-time data sync
- Batch data loading
- Transformation rules
- Data quality checks
- Conflict resolution strategies
- Version control for schemas
- Audit trail
- Rollback capability
- Performance monitoring

## Setup
### Backend
- Run with `cd backend && mvn spring-boot:run`
- Or use docker compose.

### Frontend
- Run with `cd frontend && npm run dev`
