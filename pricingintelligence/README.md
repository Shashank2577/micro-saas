# PricingIntelligence

AI pricing optimization platform. Models price elasticity from historical conversion, expansion, and churn data. Suggests optimal price points for different customer segments. Identifies where pricing creates friction vs. where customers are price-insensitive. Supports controlled pricing experiments.

## Ports

- Backend: 8174
- Frontend: Next.js standard

## Setup

1. Run Nexus Hub and infra dependencies.
2. `cd backend && mvn spring-boot:run`
3. `cd frontend && npm run dev`
