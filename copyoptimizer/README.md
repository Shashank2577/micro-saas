# CopyOptimizer

**Domain:** Marketing Intelligence
**Outcome:** Generate and score conversion copy variants before publication.

## Setup
- Backend: Java 21, Spring Boot 3.x, PostgreSQL
- Frontend: Next.js, React, Tailwind CSS

## Build Commands
- Backend: `mvn -pl copyoptimizer/backend clean verify`
- Frontend: `npm --prefix copyoptimizer/frontend run build`

## Environment Variables
- `SPRING_DATASOURCE_URL`
- `SPRING_DATASOURCE_USERNAME`
- `SPRING_DATASOURCE_PASSWORD`
- `LITELLM_BASE_URL`
