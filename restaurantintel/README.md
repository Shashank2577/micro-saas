# RestaurantIntel

AI-powered restaurant operations platform to help food & beverage operators make data-driven decisions.

## Features
- Menu Performance Analysis (Engineering Matrix)
- Predictive Ordering to reduce food waste
- Staff Scheduling Optimization based on demand
- Review Sentiment Analysis for operational insights

## Setup

### Backend
Requires Java 21 and Maven.
```bash
cd backend
mvn spring-boot:run
```

### Frontend
Requires Node.js 20.
```bash
cd frontend
npm install
npm run dev &
```
