# ProspectIQ

AI prospect intelligence platform for sales teams. Aggregates signals from funding events, hiring patterns, tech stack changes, leadership changes, news mentions, and customer reviews. Delivers prospect briefings that enable effective first calls.

## Services
- Backend: Spring Boot 3.3.5 (Port 8141)
- Frontend: Next.js 14 App Router (Port 3141)

## Setup & Running
Use the root docker-compose to start everything:
`docker compose up -d prospectiq-backend prospectiq-frontend`
