# InsuranceAI

Insurance operations platform powered by AI.

## Overview
Claims processing automation with triage and extraction. Fraud detection using behavioral anomaly patterns. Risk scoring for underwriting. Policy document extraction and comparison.

## Setup
1. Backend:
   `cd backend && mvn spring-boot:run`
2. Frontend:
   `cd frontend && npm install && npm run build && npm run dev`

## Integrations
This app integrates with:
- LiteLLM Gateway
- PostgreSQL
- Redis
