# LogisticsAI

LogisticsAI is an AI logistics intelligence platform designed to improve carrier selection, route optimization, and inventory positioning using real-time conditions and predictive models.

## Overview
- **Carrier Performance:** Monitor carriers and their on-time prediction rates.
- **Route Optimization:** Manage routes and optimize them using AI-driven systems.
- **Exception Agent:** Record incidents and get real-time AI recommendations on severity and actions.

## Getting Started

### Backend
1. Navigate to `backend/`.
2. Configure application properties (connected to local docker infrastructure by default).
3. Run `mvn spring-boot:run`.

### Frontend
1. Navigate to `frontend/`.
2. Install dependencies: `npm install`.
3. Start the dev server: `npm run dev`.
4. Access the application on `http://localhost:3000`.

### Testing
- **Backend:** `mvn clean verify`
- **Frontend:** `npm run test` (uses vitest + Testing Library)

## Documentation
- Detailed API Specs and documentation can be generated by visiting `/swagger-ui.html` on the backend server.
