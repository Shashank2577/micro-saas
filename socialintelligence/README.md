# SocialIntelligence

Multi-platform social media analytics and growth intelligence for creators.

## Features
- Unified dashboard for Instagram, TikTok, YouTube, Twitter/X, and LinkedIn
- Engagement trends and audience demographics
- Top-performing content analysis with AI
- AI-powered growth recommendations and projections

## Stack
- Backend: Spring Boot 3.3.5, PostgreSQL, cc-starter
- Frontend: Next.js 14, React, Tailwind CSS

## Getting Started

### Backend
1. Ensure PostgreSQL is running.
2. Build cross-cutting: `cd ../cross-cutting && mvn install`
3. Run backend: `cd backend && mvn spring-boot:run`

### Frontend
1. Install dependencies: `cd frontend && npm install`
2. Run frontend: `npm run dev`
