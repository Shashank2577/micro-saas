AI Architectural Blueprint

Instructions for Autonomous Agents (Jules)

  1. Scaffold: Use ./tools/scaffold-app.sh to ensure baseline dependencies.
  2. Foundation: Inherit from cc-starter. Do not reinvent RBAC, Tenancy, or Auditing.
  3. Schema: Use Liquibase for migrations. Strict schema-per-app isolation.
  4. Manifest: Every app MUST have an integration-manifest.json defining events and API contracts.
  5. Events: Emit events via WebhookEventPublisher. Integrate via Nexus Hub only.
  6. Frontend: Next.js 14, App Router, Tailwind. Use shared component library where possible.

Tech Stack Mandates

  • Java 21 / Spring Boot 3.3.5
  • TypeScript / Next.js 14
  • PostgreSQL / PGMQ / Vector
  • Keycloak (Auth) / MinIO (Storage)
