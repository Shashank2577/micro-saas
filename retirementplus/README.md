# RetirementPlus

Comprehensive retirement planning and income projection platform. Focuses on retirement income adequacy, Social Security optimization, drawdown strategies, and longevity risk management for pre-retirees and retirees.

## Quick Start

### Backend
\`\`\`bash
cd backend
mvn clean install
mvn spring-boot:run
\`\`\`

### Frontend
\`\`\`bash
cd frontend
npm install
npm run build
\`\`\`

### Tests
- Backend: \`mvn test\`
- Frontend: \`npm test\`
